---------------------- -webkit-line-clamp 是一个 不规范的属性（unsupported WebKit property），它没有出现在 css 规范草案中。 限制在一个块元素显示的文本的行数。 为了实现该效果，它需要组合其他外来的WebKit属性。常见结合属性： display: -webkit-box; 必须结合的属性 ，将对象作为弹性伸缩盒子模型显示 。 -webkit-box-orient 必须结合的属性 ，设置或检索伸缩盒对象的子元素的排列方式 。 text-overflow，可以用来多行文本的情况下，用省略号“…”隐藏超出范围的文本 。 只在webkit内核的浏览器有用，可以用在MOB端页面的开发
